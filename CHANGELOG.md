## [0.3.4] — 2026-07-15

### Adicionado

- Publicação inicial dos 19 componentes do Design System DetraSoft.
- Suporte a Angular 17+ como peer dependency.
- 2 locales para Datepicker (PT-BR, EN).
- ToastService com signal-based state.
- CDK Overlay para Dropdown, Autocomplete, Search, SearchModal, Dialog.
- CSS puro com variáveis de tema (`--badge-color`, etc).
